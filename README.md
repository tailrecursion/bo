# BO

A minimal implementation of LightTable's BOT Architecture.

# Patterns

* Object Manager
* Interceptor Pattern
* Entity Systems

BO can be used to glue different parts within the application with behaviors.  

It can help augment the Data Flow in javelin with

* Data Store
* Control Flow

# Testing

```
$ lein cljsbuild auto
$ node target/test.js
```
